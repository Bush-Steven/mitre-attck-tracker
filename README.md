# 🛡️ MITRE ATT&CK Tracker (Personal SOC Intelligence Tool)

A lightweight **MITRE ATT&CK tracking and analysis platform** built with Flask and SQLite.  
Designed for cybersecurity learners, SOC analysts, and blue team practitioners to map, track, and analyze attack techniques (TTPs) based on the MITRE ATT&CK framework.

---

## 📌 Project Overview

This tool allows users to log and monitor simulated or real security incidents and map them to MITRE ATT&CK techniques.

It helps answer questions like:
- What attack techniques am I observing or simulating most often?
- Which MITRE tactics are most common in my environment/lab?
- How can I visualize my threat exposure?

---

## 🎯 Key Features

- ➕ Add attack records manually
- 🧠 Map incidents to MITRE ATT&CK Technique IDs (e.g., T1566)
- 📊 SOC-style dashboard with severity analytics
- 🗂️ Categorization by tactic (Initial Access, Execution, etc.)
- 💾 SQLite-based local storage (lightweight & fast)
- 🌐 Web-based interface (Flask)
